{
    "id 01": {
        "title": "Hello",
        "type": "error",
        "content": "This is the content of site message."
    },
    "02": {
        "title": "Hello",
        "content": "This is the content of site message."
    },
    "03": {
        "content": "This is the content of site message."
    }
}
  • 消息 ID（id 01、02）：一个任意唯一的字符串。ID 中的空格会被替换为 -。
  • title（可选）：消息标题，任意字符串。
  • type（可选）：消息样式，可用的有 default（默认）、info、warning、danger/error。
  • content：消息正文内容，任意字符串，支持 HTML。
所有消息都将会在最后一个消息被关闭后 14 天再次出现。
