Git push 出错

问题描述

执行 git push 命令出现以下错误提示：

1kex_exchange_identification: read: Connection reset by peer
2Connection reset by 20.205.243.166 port 22
3fatal: 无法读取远程仓库。
4
5请确认您有正确的访问权限并且仓库存在。

解决方法

1mv ~/.ssh/known_hosts ~/.ssh/known_hosts.old
2# 或者删除 ~/.ssh/known_hosts 中的某一行（如果能确定是哪行）
3sudo systemctl restart sshdgenkeys.service
4git push

出现以下提示：

1The authenticity of host 'github.com (20.205.243.166)' can't be established.
2<密钥类型> key fingerprint is <公钥>.
3This key is not known by any other names
4Are you sure you want to continue connecting (yes/no/[fingerprint])?

输入 yes，回车，出现下面的提示，表明 github.com (20.205.243.166) 被添加到 ~/.ssh/known_hosts 文件中。

1Warning: Permanently added 'github.com' (<密钥类型>) to the list of known hosts.

提交成功。
