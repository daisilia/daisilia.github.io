正常水、钠平衡

体液（body fluid）
体内的水和溶解在其中的物质。
  • 细胞内液：K^+^, HPO~4~^2-^
  • 细胞外液：Na^+^, Cl^-^
    • 组织间液
    • 血浆：蛋白质含量高，正常渗透压范围 ==290~310mmol/L==，血清 Na^+^ 浓度正常范围 ==135~145mmol/L==

渴感

渴感（thirst）
由水分不足或摄入盐过多导致细胞外液渗透压升高，刺激渗透压感受器（下丘脑视上核和视旁核）而产生。
