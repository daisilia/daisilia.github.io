Vue 组件化

注意事项

  1. 组件初始化所用的配置对象中没有 el 选项。
  2. 组件中的 data 配置项中的“数据”要写成函数。

关于组件起名

组件名推荐写法：

  1. 一个单词：school、School。
  2. 多个单词：myschool、"my-school"、MySchool（需要脚手架）。

尽量回避 HTML 中内置的标签名（注意 HTML 忽略大小写）。

在组件配置对象中使用 name 配置项可指定组件在 Vue 开发工具中显示的名称。

基本语法

 1// 1. 创建组件
 2const childComponent = Vue.extend({
 3    template: `
 4        <div class="child">
 5            姓名：<input type="text" name="姓名" v-model="auther.name">
 6            邮箱：<input type="text" name="邮箱" v-model="auther.email">
 7        </div>
 8    `,
 9    data: function(){
10        return {
11            auther: {
12                name: 'melody',
13                email: 'melody@163.com',
14            },
15            // ...
16        }
17    },
18    // ...
19})
20
21new Vue({
22    el: '#root',
23    // 2. 注册组件（局部注册）
24    components: {
25        // childComponent,
26        child: childComponent,
27    },
28})

然后在 HTML 中添加组件标签：

1<div id="root">
2    <!-- 挂载组件 -->
3    <child></child>
4    <child/><!-- 注意 Vue 脚手架才能正确处理好多个自闭合标签 -->
5</div>
注册全局组件

注册全局组件的方法与注册全局过滤器、全局自定义指令的方法类似，使用 Vue.component 方法。

Vue 也可以直接用组件配置对象作为 components 中的组件的值，即上述对应代码可简写为：

1const childComponent = {
2    template: /* ... */,
3    data: function(){
4        return {
5            // ...
6        }
7    },
8    // ...
9}

组件嵌套

 1const coc = {
 2    template: `
 3        <div class="child">
 4            <p>信息：{{mes}}</p>
 5        </div>
 6    `,
 7    data: function(){
 8        return {
 9            mes: "hello",
10            // ...
11        }
12    },
13}
14
15// 1. 创建组件
16const childComponent = Vue.extend({
17    // ...
18    components: {
19        'child-of-child': coc,
20    },
21    // ...
22})
